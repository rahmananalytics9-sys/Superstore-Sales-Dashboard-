# Superstore-Sales-Dashboard-
This project analyzes retail sales data from a Superstore dataset to understand sales performance, customer behavior, and regional trends. The dashboard provides business insights that help identify high-performing products, profitable segments, and seasonal trends.
Tools Used
•	Power BI
•	SQL
•	Excel
•	Data Visualization
Dataset Features
The dataset includes the following fields:
•	Order ID
•	Order Date
•	Ship Date
•	Ship Mode
•	Customer ID
•	Customer Name
•	Segment
•	Country
•	City
•	State
•	Region
•	Product ID
•	Category
•	Sub-Category
•	Product Name
•	Sales
•	Quantity
•	Profit
•	Returns
•	Payment Mode
Key KPIs
•	Total Sales
•	Total Profit
•	Total Orders
•	Total Quantity Sold
•	Profit Margin
Dashboard Insights
Sales by Category
Technology generates the highest revenue, followed by Furniture and Office Supplies.
Monthly Sales Trend
Sales peak during November and December due to holiday demand.
Sales by Sub-Category
Phones and Chairs contribute significantly to total revenue.
Sales by Segment
The Consumer segment accounts for the largest share of sales.
Sales by Ship Mode
Standard Class is the most commonly used shipping method.
Regional Sales
The West region contributes the highest sales while the South region has the lowest.
Profit Trend
Profit follows seasonal sales patterns with peak values in the fourth quarter.
Payment Mode Analysis
Credit card transactions dominate the payment methods used by customers.
Conclusion
The Superstore Sales Dashboard provides valuable insights into sales performance, customer segments, and product profitability, helping businesses make data-driven decisions.
